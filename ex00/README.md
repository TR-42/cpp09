# プログラムの入力仕様について

## データベースファイル

DBファイルで使用するcsv形式の定義は、以下の通りとする。

- ファイル名: `data.csv`
- ヘッダ行: あり (先頭1行、かつカラム名と一致させる)
- 区切り文字: `,`
- 区切り文字周辺のスペース: 不可
- 文字コード: ASCII
- 改行コード: LF (`getline`の実装に依る)
- 末尾の空行: 任意 (あってもなくてもよい)
- 途中の空行: 禁止

|カラム名|データ型|制約|
|---|---|---|
|`date`|`Date`|`YYYY-MM-DD`、かつ昇順|
|`exchange_rate`|`unsigned float`|`0.0`以上|
